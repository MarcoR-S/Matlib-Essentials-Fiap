# 📊 Data Visualization com Matplotlib

Este repositório contém uma coleção de scripts em Python para geração dos principais tipos de gráficos utilizados em Ciência de Dados.


## 🚀 Gráficos Implementados
- **Line Chart**: Monitoramento de tendências temporais.
- **Scatter Plot**: Análise de correlação entre variáveis.
- **Histogram**: Visualização de distribuições estatísticas.
- **Bar Chart**: Comparação de performance entre categorias.
- **Pie Chart**: Representação de proporções e Market Share.

## 🛠️ Tecnologias Utilizadas
- Python 3.x
- Matplotlib
- NumPy

## 📖 Como usar
1. Clone o repositório: `git clone https://github.com/SEU_USUARIO/matplotlib-essentials`
2. Instale as dependências: `pip install matplotlib numpy`
3. Execute o arquivo principal: `python main.py`
